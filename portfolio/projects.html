First Project

This short story captures one of my memorable trips, showcasing my initial steps into applying HTML in a real project. I focused on using essential HTML elements to structure my webpage. Specifically, I utilized a header to introduce the title and set the tone for the page, a nav section to create a simple navigation menu, and an unordered list (ul) to neatly present key highlights from the trip. I also included an image (img) element to display a scenic photo from the journey, giving the page a visual appeal. Finally, I embedded a video to share a more immersive experience with viewers.
